在github上，Settings->SSH and GPG keys点击New SSH Key按钮，将id_rsa.pub公钥数据复制到Key文本框，保存完成。

修改配置文件（单机器多个ssh重点）

检查用户/.ssh目录下，是否有config文件，若是没有，新建名称为config的文件(注：该文件没有文件类型后缀)。 编辑文件，添加以下内容：

##别名，github默认为：github.com
Host github_wu.com

##数组名称
HostName github.com

##账号
User git
PreferredAuthentications publickey

##指定私钥文件，默认为~/.ssh/id_rsa
IdentityFile ~/.ssh/id_rsa.wu

~/.ssh/id_rsa.wu为生成ssh时，个人指定的文件路径，默认名称为~/.ssh/id_rsa。
